# ⚡ RapidLabs Print Quote Calculator

> **Instant wide-format print quotes — no forms, no waiting.**  
> Get your banner, vehicle wrap, acrylic sign, vinyl decal, canvas, or backlit display quote in seconds.

🔗 **Live site:** https://malcmitch.github.io/rapidlabs-print-calculator/

---

## What It Does

Drop in your dimensions, pick your substrate, choose a finish, and hit **Calculate** — you get a real-time estimate with price/sq ft, total range, turnaround time, and which Roland printer handles your job.

### Print Types Supported
| Type | Substrate | Printer |
|------|-----------|---------|
| Banners | 13oz Banner / Mesh | Roland VG3-640 (Eco-Solvent) |
| Vehicle Decals / Wraps | Adhesive Vinyl | Roland VG3-640 |
| Signs & Plaques | Acrylic, Foam Board, MDF | Roland LG-640 (UV-LED) |
| Canvas Prints | Canvas | Roland VG3-640 |
| Backlit Displays | Backlit Film | Roland VG3-640 |
| Floor Graphics | Anti-Slip Floor Vinyl | Roland VG3-640 |

### Features
- Real-time quote with low/high range
- Quantity-based volume discounts
- Rush turnaround toggle (+30%)
- Finish selector (matte, gloss, satin, laminate)
- Unit toggle (inches / feet)
- Zero dependencies — single HTML file, runs offline
